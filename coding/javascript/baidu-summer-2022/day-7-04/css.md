# CSS基础
## CSS选择器
###  1. 类型、类和ID选择器
```html
p {
    color:blue;
}

.class1 {
    color: red;
}

#id1 {
    color: green;
}
```
ID选择器 > 类选择器 > 类型选择器
### 2. 标签属性选择器
```html
img[src] {
    width: 100px;
}
```
### 3. 伪类和伪元素
```html
a:hover {
    color: red;
}
```

### 4. 关系选择器

```html
div span {
    color: red;
}
```
