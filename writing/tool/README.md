# 记录笔记的常用工具

## [docsify](writing/tool/docsify.md)

> docsify是一个轻量级的笔记框架，直接使用markdown渲染，无需编译成html代码。

## [dot](writing/tool/dot.md)

> dot是一个简单的描绘图形的脚本语言，便于绘制流程图、类图，用户更关注于逻辑关系，实现所思即所得。
